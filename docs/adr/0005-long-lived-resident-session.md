# ADR-0005: 住人のセッションは長命にする

- ステータス: Accepted（P2で実証）
- 日付: 2026-06-26
- 関連: ADR-0006

## 背景 / 課題
ユーザーが茶々に話しかけられるようにする（双方向）と、「さっきの夕日」等の **文脈継続が必要**になる。

## 決定
茶々のACPセッションは **session/new を起動時1回だけ**。環境つぶやきもユーザー発話も同一セッションに積み、文脈を継続する。

## 検討した代替案
- **1ターン1セッション**: 文脈が切れて「さっき」が通じない。双方向と両立しない。却下。

## 影響 / 帰結
- P2で実証（「雨、まだやんでへんな」等、前ティックを踏まえた発話が出た）。
- initialize 応答に resume/fork/list があり、長命＋要所 fork が現実的。

## 備考
- 長命ゆえコンテキスト肥大が将来課題（compaction戦略、forkの閾値）。spec Open Question。
