# ADR-0009: 不透明な隅ウィンドウにする（透過しない・背景ごと描く）

- ステータス: Accepted
- 日付: 2026-06-26
- 関連: ADR-0010

## 背景 / 課題
デスクトップの隅に「住んでる」感を出したい。当初はウィンドウ透過（frameless + transparent）を想定。

## 決定
透過に頼らず、**縁側の背景（空・障子・板の間）ごと不透明に描いた小窓**を、frameless + on_top で画面隅に置く。

## 検討した代替案
- **transparent=True で背景透過**: Windows + EdgeChromium でフチのにじみ等、完成度が環境依存で「賭け」になる。市販の類似デスクトップアプリも透過せず背景ごと描いていた。→ 透過は捨てる。

## 影響 / 帰結
- 一番怖かった透過の不確実性がゼロに。技術的に堅い道が見た目的にも正解。
- 段階導入: 枠あり → frameless → （必要なら）透過、と外せる。

## 備考
- 既存アプリのアセット流用は不可。構造（隅に箱庭を置く）のみ学び、絵はオリジナル（ADR-0010）。
